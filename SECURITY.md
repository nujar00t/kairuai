# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ |

## Reporting a Vulnerability

Do **not** open a public issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](https://github.com/KairuAI/kairuai/security/advisories/new).

We will respond within 48 hours.

## Important Notes

- Never commit your `.env` file — it is in `.gitignore`
- `SOLANA_PRIVATE_KEY` gives full control of your wallet — keep it secret
- The trading agent executes real transactions — test with small amounts first
- KairuAI predictions are not financial advice
