# Changelog

All notable changes to KairuAI are documented here.

## [1.0.0] - 2025-03-30

### Added
- Prediction Agent — AI-powered price prediction using Claude + technical indicators
- Signal Agent — sentiment analysis from Twitter/X + crypto news
- Trading Agent (TypeScript) — Solana DEX swaps via Jupiter Aggregator
- Polymarket Agent (TypeScript) — prediction market intelligence and opportunity scanner
- TypeScript SDK — shared client library for KairuAI agents
- Unified Python CLI with interactive menu (`kairu.py`)
- GitHub Actions CI — Python syntax check + TypeScript build check
- Makefile for streamlined setup and agent execution
- One-line install script (`install.sh`)
